.frani{}

Frani es una hoja de estilo CSS elaborada para la construcción de sitios web responsivos. Su filosofía radica en ser minimalista y con una sintaxis simple y de fácil aprendizaje, que es altamente flexible y de rápida implementación.

Instalación

La instalación es tan simple como colocar la siguiente etiqueta en la cabeza de tu documento html:

<link rel="stylesheet" src="https://frani.beq.mx/frani.min.css">

Otra opcion es descargar el archico CSS de AQUÍ. Siempre recomendamos utilizar frani como la primer hoja de estilo en tu documento.

Como se usa

Frani esta dividido en dos partes, la cuadricula y el formato. La parte del formato solo son unas pocas opciones que ayudan a dar formato al texto y contenedores. Le recomendamos verificar la documentación de esa sección para conocer todas las opciones.
